# 小艸西手作甜點訂購表單 dessert_order

這是一個使用 Flask跟JS 製作的簡單線上甜點訂購頁面，供作品展示使用。
使用者可透過前端介面選購甜點、填寫基本資訊，並將資料送出至 Google 表單。

---

# 使用技術

- HTML / CSS / JavaScript
- Python Flask
- Git / GitHub
---
